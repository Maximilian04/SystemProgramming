QuadrEquation

Project for solving quadratic equations, with documentation and testing system

The project has its own doxygen documentation: badlink

Features

  • Run make with makefile. Standart tests will be done automaticly.
  • Run build/QuadrEquation.exe for solve equation via typing coefficients in terminal.
  • Run build/tests.exe without cmd arguments for run standart tests.
  • Run build/tests.exe with file name as cmd argument for run your own tests. (example: myTest.txt)
  • Run doxygen with Doxyfile for genarate documentation. Ready-to-use script runDoxygen.bat.

Tools

  • C/C++ compiler: GCC (x86_64-posix-seh-rev0, Built by MinGW-W64 project) 8.1.0

Documentation
